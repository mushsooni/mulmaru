# 물마루 | Mulmaru
**굵고** _부드러운_ 한글 픽셀 폰트
![01_title](images/01_title.png)



## 소개
**물마루**는 도트 게임의 한국 현지화 패치 과정에서 제작된 폰트입니다.

평소 한글패치를 제작하면서 원작 게임과 비슷한 스타일의 폰트를 찾는 데 어려움을 겪곤 했습니다.  
둥근모<sup>[[1]](https://cactus.tistory.com/193) [[2]](https://neodgm.dalgona.dev/)</sup> 등 16픽셀 폰트보다는 작고, [갈무리](https://quiple.dev/galmuri)보다는 두꺼우면서도 가독성이 좋은 폰트가 없다는 점이 늘 아쉬웠습니다.

그러던 중 exqt님의 [Pixel Font Maker](https://exqt.me/posts/pixel-font-maker)라는 프로그램의 존재를 발견하고, 제가 그리던 폰트를 직접 제작하게 되었습니다.
![02_intro](images/02_intro.png)
물마루는 갈무리11과 같은 크기이면서도, 둥근모처럼 세로획이 굵고 모서리가 둥근 산세리프 폰트로 제작되었습니다.

한글 자형은 [Yoon px 아케이드](https://font.co.kr/collection/sub?family_idx=10627)의 디자인을 주로 참고하였고, [10x6x4벌식](https://github.com/exqt/pixel-font-maker?tab=readme-ov-file#minzkn님-10x6x4벌식-글꼴) 조합형 템플릿을 바탕으로 리디자인 및 변형을 거쳐 만들어졌습니다.

## 내려받기
**[최신 버전 다운로드](https://github.com/mushsooni/mulmaru/releases/latest)**
![04_mono](images/04_mono.png)
- **물마루**는 가변폭 폰트, **물마루 Mono**는 고정폭 폰트입니다.
- 물마루와 물마루 Mono는 **12px** (9pt) 크기 또는 그 배수에서 가장 명확하게 표시됩니다.
- .pfp 파일은 [Pixel Font Maker](https://exqt.github.io/pixel-font-maker/)에서 불러오고 수정할 수 있는 원본 json 파일입니다.

## 면책 조항
이 프로젝트는 둥근모, 갈무리, Yoon px 아케이드 등 상기 언급된 폰트의 원작자와 어떠한 관련도 없습니다.
